# Banner Commands

Show a title screen before starting the story.

## Banner (Title only)

- Syntax: `banner {banner title}`
- Example:

  ```txt
  banner 'New Products'
  ```

## Banner (With subtitles)

- Syntax: `banner2 {banner subtitle} {banner primary title}`
- Example:

  ```txt
  banner2 "It\'s subtitle" 'It\'s Big Title'
  ```

# Cover Commands

## Set Overlay Image

{nameId} is obtained when you preload an overlay image. See: [Load Command: Load Background Scenery](./1-Command_load.md#load-overlay-image "Load Command: Load Overlay Image")

- Syntax: `cover set {nameId}`
- Example:

  ```txt
  cover set player
  ```

## Show Overlay Image

- Syntax: `cover show`

## Hide Overlay Image

- Syntax: `cover hide`

# 左上角小标题 label

- 命令：label {text}
- 例子：label TestClass

# 横幅 banner

- 命令：banner {text}
- 例子：banner Test

# 文本框 txt

- 命令：txt {name} {group} "{content}"
- 例子：txt 日步美 补课部 "啊.哈哈..."
- 内容支持富文本

# 文字大小 txtSize 

- 命令：txtSize {size}
- 例子：txtSize small
- size [big,medium,small]

# 选择按钮 button

- 命令：button {text1} {targetName1} {text2} {targetName2} {text3} {targetName3}
- 例子：button 人物(Button1) tSpr 图片(Button2) tBg 声音(Button3) tSound
